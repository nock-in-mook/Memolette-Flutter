描画では:
- `displayAngle = rawIndex * itemAngle - rotation`
- セクターの中央角度 = `(180 - displayAngle) * pi / 180`
- つまりセクター中央の角度（ラジアン、Canvas座標）= `(180 - displayAngle) * pi / 180`

- タッチ位置からCanvasのatan2で角度取得
- `angleRad = atan2(dy, dx)` （これはCanvas座標系そのまま）
- `(180 - displayAngle) * pi / 180 = angleRad`

- `rawIndex = (displayAngle + rotation) / itemAngle`
