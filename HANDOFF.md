## 技術メモ
- **ビルド回避策**: `/tmp/memolette-run` に rsync してから build（Google Driveで codesign エラー回避）
- **2台目Mac**: rsync 後に `flutter pub get && cd ios && pod install` が必要
- **シミュレータ**: 新マシン `EA0AC2A3-C01C-4A0E-9FC5-DC0D503748AB` (旧 `29B0ACCA-...` は別マシン)
- **実機**: iPhone 15 Pro Max `30A153A2-9507-5499-8B3D-341320DA2AB3`（devicectl）/ `00008130-0006252E2E40001C`（flutter）
- **ホットリロード**: シミュレータは fifo + `echo r > /tmp/flutter-sim-stdin` で。Stack構造変更等は hot restart (R) 必要
- **実機インストール**: `flutter build ios --release` → `xcrun devicectl device install app` が安定（`flutter run --release` だと install 失敗することあり）
- **メモ背景色**: `MemoBgColors.getColor(index)` で `Color.lerp(base, Colors.white, 0.18)` で白寄せ
- **画面端スワイプ領域**: 検索バー foreground 時にフォルダ本体を消す条件 `_isSearchFocused && !_isSearchActive`
- **空メモ判定（最終）**: titleEmpty && contentEmpty（色は見ない）。色付きメモはフォーカス外れだけスキップ
